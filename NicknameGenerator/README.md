# 🎮 재미있는 닉네임 생성기 API (.NET 10)

.NET 10을 사용한 초고속 로컬 닉네임 생성 API입니다. 외부 API 호출 없이 1ms 이내에 재미있고 독특한 닉네임을 생성합니다!

## ✨ 주요 기능

- ⚡ **초고속**: 1ms 이내 응답 (외부 API 대비 1000배 빠름)
- 💰 **완전 무료**: API 비용 없음
- 🎲 **무한한 조합**: 수백만 가지 닉네임 생성 가능
- 😂 **재미있는 결과**: 웃긴 조합과 밈 요소 포함
- 🔒 **안정적**: 네트워크 오류 없음
- 📦 **오프라인**: 인터넷 연결 불필요

## 📋 필수 요구사항

- .NET 10 SDK

## 🚀 설치 및 실행

### 1. 프로젝트 디렉토리로 이동
```powershell
cd NicknameGenerator
```

### 2. 애플리케이션 실행
```powershell
dotnet run
```

서버가 `http://localhost:5000`에서 시작됩니다.

## 🎮 API 사용 방법

### 닉네임 생성
```http
GET http://localhost:5000/api/nickname
```

**응답 예시:**
```json
{
  "nickname": "불타는라면God777"
}
```

### cURL 예제
```bash
curl http://localhost:5000/api/nickname

### PowerShell 예제

Invoke-RestMethod -Uri http://localhost:5000/api/nickname
```

### 브라우저에서 테스트
```

```

## 🎨 닉네임 생성 알고리즘

### 8가지 생성 스타일

1. **기본형**: `형용사 + 명사`
   - 예: `행복한고양이`, `폭발하는치킨`

2. **숫자형**: `형용사 + 명사 + 숫자`
   - 예: `용감한전사42`, `발효된똥42069`

3. **접미사형**: `형용사 + 명사 + 접미사`
   - 예: `전설의변기님`, `불타는라면God`

4. **더블형**: `명사 + 명사`
   - 예: `좀비뱀파이어`, `치킨피자`

5. **복합형**: `형용사 + 명사 + 접미사 + 숫자`
   - 예: `멋진마법사킹7823`

6. **재미형**: `접두사 + 명사 + 접미사`
   - 예: `슈퍼방귀ㅋㅋ`, `메가치킨왕`

7. **트리플형**: `형용사 + 명사 + 명사`
   - 예: `황금떡볶이용사`, `어둠의슬라임전사`

8. **완전랜덤**: `모든 요소 랜덤 조합`
   - 예: `울트라사기친휴지할배9999`
