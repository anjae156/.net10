# 🎮 재미있는 닉네임 생성기 API (.NET 8)

.NET 8을 사용한 초고속 로컬 닉네임 생성 API입니다. 외부 API 호출 없이 1ms 이내에 재미있고 독특한 닉네임을 생성합니다!

## ✨ 주요 기능

- ⚡ **초고속**: 1ms 이내 응답 (외부 API 대비 1000배 빠름)
- 💰 **완전 무료**: API 비용 없음
- 🎲 **무한한 조합**: 수백만 가지 닉네임 생성 가능
- 😂 **재미있는 결과**: 웃긴 조합과 밈 요소 포함
- 📦 **배치 생성**: 한 번에 여러 개의 닉네임 생성 가능
- 🎨 **스타일 선택**: 원하는 패턴(웃긴 거, 판타지 등) 지정 가능
- 📏 **길이 제한**: 닉네임 최대 길이 설정 가능
- 📄 **Swagger UI**: 웹 브라우저에서 API 테스트 가능

- .NET 8 SDK

## 🚀 설치 및 실행

### 1. 프로젝트 디렉토리로 이동
```powershell
cd NicknameGenerator
```

### 2. 애플리케이션 실행
```powershell
dotnet run
```

서버가 `http://localhost:5000`에서 시작됩니다.

## 🎮 API 사용 방법

### 기본 생성 (랜덤 1개)
```http
GET http://localhost:5000/api/nickname
```

### 고급 생성 (파라미터 사용)
```http
GET http://localhost:5000/api/nickname?count=5&style=funny&maxLength=10
```

- **count**: 생성할 닉네임 개수 (기본값: 1, 최대: 100)
- **style**: 생성 스타일 (basic, number, suffix, double, complex, funny, triple, crazy)
- **maxLength**: 닉네임 최대 길이 (선택 사항)

**응답 예시:**
```json
{
  "count": 5,
  "nicknames": [
    "슈퍼방귀ㅋㅋ",
    "메가치킨왕",
    "진짜변기님",
    "초대형똥God",
    "울트라휴지맨"
  ]
}
```

### Swagger UI (API 문서)
브라우저에서 아래 주소로 접속하면 API를 테스트할 수 있습니다:
```
http://localhost:5000/swagger
```

## 🎨 닉네임 생성 스타일 (`style` 옵션)

1. **basic**: `형용사 + 명사` (예: 행복한고양이)
2. **number**: `형용사 + 명사 + 숫자` (예: 용감한전사42)
3. **suffix**: `형용사 + 명사 + 접미사` (예: 전설의변기님)
4. **double**: `명사 + 명사` (예: 좀비뱀파이어)
5. **complex**: `형용사 + 명사 + 접미사 + 숫자` (예: 멋진마법사킹7823)
6. **funny**: `접두사 + 명사 + 접미사` (예: 슈퍼방귀ㅋㅋ)
7. **triple**: `형용사 + 명사 + 명사` (예: 황금떡볶이용사)
8. **crazy**: `완전 랜덤 조합` (예: 울트라사기친휴지할배9999)
